Frequently Asked Questions

Do I need to install Python?

No! The standalone executable bundles Python, PySide6, and the Gemini SDK. Just double-click the .exe file to run it.

Which applications are supported?

LingoGlide runs everywhere on Windows. It works inside Notepad, WhatsApp Desktop, Slack, ChatGPT, LinkedIn, Microsoft Word, VS Code, and browser text fields.

How does the app selection work?

LingoGlide dynamically reads the foreground app process name and title. It uses box selection (Ctrl+A) in browsers and chat clients, and line selection (Shift+Home) in coding and text editors.

Is it safe to use?

Yes. Your API key and input text are kept entirely on your computer. It never communicates with any server other than directly to the Google Gemini API endpoint.
